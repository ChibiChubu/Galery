# AI Video Gallery

Static preview gallery for ChibiChubu AI video work.

## Local preview

Open `index.html` directly in a browser, or run a simple local server:

```powershell
python -m http.server 5173
```

Then visit `http://localhost:5173`.

## GitHub Pages

After pushing to GitHub, enable GitHub Pages from the repository settings and set the source to the `main` branch root.
